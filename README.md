## Run Locally

Clone the Project

```bash
  git clone https://github.com/Nafiz-Anam/hbut_cms_server
```

Go to the project directory

```bash
  cd hbut_cms_server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
